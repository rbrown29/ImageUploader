# ImageUploader

### This is jQuery UI to build a sophisticated interface, implement a file drag-and-drop facility, and use AJAX to store the images in a database.

* HTML
* CSS
* JAVASCRIPT
* JQUERY UI
* AJAX
* PHP
* MySQL
* XAMPP
